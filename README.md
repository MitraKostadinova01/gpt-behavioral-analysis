# Python Scripts for Testing ChatGPT Behavior in Various Work Positions Across Different Geographic Locations

This repository, named `gpt-behavioral-analysis`, contains Python scripts designed to evaluate how ChatGPT models behave when simulating a representative role from IT industry. The scripts interact with ChatGPT, posing various scenarios to assess the model's responses and effectiveness in this context.

Additionally, a script for plotting graphs from the obtained results is included. All data are presented in CSV files, categorized by the models used.
